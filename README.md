#  Matsu Worship Interactive Installation — 神鳥尋寶記：解開東莒寶藏的謎團 / Unlocking the Secrets of Dongju

## 繁體中文

一款結合馬祖莒光鄉傳統文化與自然生態的 2D 像素風格橫向捲軸冒險遊戲與互動裝置。

### 🚀 專案展示
* 📸 **專案海報 / 主視覺**：

<img width="1090" height="1502" alt="神鳥尋寶記海報" src="https://github.com/user-attachments/assets/d13e1116-eef0-4073-8a54-e5210920f001" />

### 💡 專案簡介
藉由 2D 像素卷軸遊戲，讓玩家在收集大埔石刻幻化成的馬祖原生種彼岸花與莒光五寶的過程，引領體驗莒光鄉的傳統文化與自然生態。透過虛擬的冒險故事與精緻的像素美術，將馬祖珍貴的文化資產與生態特色生動地呈現給大眾。

### 🎮 遊戲核心元素
* **大埔石刻：** 遊戲的核心主線與歷史解謎元素。
* **馬祖原生種彼岸花：** 作為遊戲中重要的收集要素。
* **莒光五寶：** 結合地方文史的特色道具與關卡機制。

## English

A 2D pixel art side-scrolling adventure game that seamlessly integrates the traditional culture and natural ecology of Juguang Township, Matsu.

### 🚀 Live Visuals
* 📸 **Project Poster / Key Visual**:

<img width="1090" height="1502" alt="Mythical Bird Treasure Hunt Poster" src="https://github.com/user-attachments/assets/d13e1116-eef0-4073-8a54-e5210920f001" />

### 💡 About The Project
Through an immersive 2D pixel-art side-scrolling experience, players embark on a journey to collect the native *Lycoris radiata* (spider lily) of Matsu and the renowned "Five Treasures of Juguang," which are mystically transformed from the historic Dapu Inscription. This interactive game guides players to discover the rich traditional heritage and natural ecology of Juguang Township, blending regional history with modern gameplay.

### 🎮 Core Gameplay Elements
* **Dapu Inscription:** The central storyline and historical puzzle-solving element of the game.
* **Matsu Native Lycoris:** Serves as a vital collectible item scattered throughout the adventure.
* **The Five Treasures of Juguang:** Unique cultural props and level mechanics deeply rooted in local history.
